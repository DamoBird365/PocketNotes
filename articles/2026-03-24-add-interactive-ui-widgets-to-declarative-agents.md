## Summary

The article discusses how to integrate interactive UI widgets into declarative agents using a Model Context Protocol (MCP) server and the OpenAI Apps SDK within Microsoft 365 Copilot. It outlines the prerequisites, best practices, and steps to create, update, and test these agents, emphasizing the importance of checking API availability and implementing proper authentication.

## Key Points

- Interactive UI widgets can be added to declarative agents via MCP server actions.
- Microsoft 365 Copilot supports UI widgets created with the OpenAI Apps SDK.
- Prerequisites include a remote MCP server, Visual Studio Code, and the latest Microsoft 365 Agents Toolkit.
