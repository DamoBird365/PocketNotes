📌 PocketNotes Settings

Configure your GitHub connection to start saving articles.

🔑 GitHub Authentication

Create a fine-grained personal access token with:
• Repository access: Only your PocketNotes repo
• Permissions: Actions (Read & Write) + Contents (Read & Write)
Stored locally in your browser — never sent anywhere except GitHub's API

📦 Repository

Your GitHub username or organisation
The name of your PocketNotes repo

🔍 Copilot Spaces (Optional)

Create a Copilot Space pointing at your articles/ folder for conversational search over your saved articles. See the setup guide.
Paste the URL of your Space (e.g. https://github.com/copilot/spaces/your-space-id)
